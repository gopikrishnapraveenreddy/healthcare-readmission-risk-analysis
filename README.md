# Healthcare Readmission Risk Analysis

## Project Overview

Healthcare organizations face significant financial and operational challenges due to patient readmissions. This project analyzes hospital readmission patterns, patient utilization behavior, and healthcare cost exposure using Python, SQL, and Tableau.

The objective is to identify high-risk patient segments, understand key readmission drivers, and provide actionable insights through executive-level dashboards.

---

## Business Objectives

* Analyze 30-day patient readmissions
* Identify high-risk patient populations
* Measure healthcare utilization patterns
* Evaluate patient complexity levels
* Quantify readmission cost exposure
* Support data-driven healthcare decision making

## Dashboards

### Dashboard 1: Healthcare Readmission & Patient Utilization Analysis

Key KPIs:

* Total Encounters
* Unique Patients
* 30-Day Readmission Rate
* Average Length of Stay
* High Utilization Rate
* Readmission Cost Exposure

### Dashboard 2: Healthcare Readmission Analysis

Key Insights:

* Patients aged 80+ exhibit the highest readmission risk.
* Emergency admissions show the highest readmission rates.
* Diagnosis groups significantly influence readmission outcomes.
* Discharge disposition impacts readmission likelihood.

### Dashboard 3: Healthcare Resource Utilization & Cost Exposure Analysis

Key Insights:

* Older patients generate higher healthcare cost exposure.
* Patient complexity increases with age.
* Resource utilization varies significantly across patient segments.
* Certain diagnosis groups require longer average hospital stays.
